# GCP Sales Data Pipeline

## 🚀 Overview
End-to-end data pipeline using GCP services:
- GCS (storage)
- Dataproc (PySpark)
- Airflow (Composer)
- BigQuery (Data Warehouse)

## 📊 Architecture
CSV → GCS → Dataproc → BigQuery → Looker Studio

## 🛠 Tech Stack
- Python
- PySpark
- GCP
- Airflow

## ▶️ Steps to Run
1. Upload CSV to GCS
2. Upload PySpark job to GCS
3. Create Dataproc cluster
4. Deploy Airflow DAG
5. Run pipeline
6. Query data in BigQuery

## 📌 Output
- Customer total spending table
- Dashboard in Looker Studio
